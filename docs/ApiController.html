ApiController

ApiController

Controller responsible for the autogenerated API: /admin_root/api/..., where admin_root is the rootPath given in AdminBroOptions.

The best way to utilise it is to use ApiClient on the frontend.

Available API endpoints

Endpoint Method Description
.../api/resources/{resourceId} ApiController#index List of records
.../api/resources/{resourceId}/search/{query} ApiController#search Search record by query string
.../api/resources/{resourceId}/actions/{action} ApiController#resourceAction Perform cusomised resource action
.../api/resources/{resourceId}/records/{recordId}/{action} ApiController#recordAction Perform cusomised record action
.../api/dashboard ApiController#dashboard Perform cusomised dashboard action
